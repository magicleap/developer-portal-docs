This article explains what action the ML2 Compute Pack's LED patterns signify.

## Beltpack LED Patterns

|                                            |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| ------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Event                                      | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| ![](/img/releases/led-images/image002.jpg) | <h3>Powering Up</h3><br></br>LEDs 1-4 will be all turned on sequentially.<br></br>LED 1 will turn off.<br></br>LED 5 will turn on.<br></br>LEDs 2-5 will turn off sequentially.<br></br>It’ll be a looping pattern. It’ll be stopped when the icon grid comes up on the wearable.<br></br>This pattern will also be displayed when the device is in fastboot mode.                                                                                                                                                                                                                                                    |
| ![](/img/releases/led-images/image004.gif) | <h3>Powering Down</h3><br></br>LEDS 1-5 will all be on white. LED 5 will turn off, followed by LED 4, then 3 through 1 sequentially.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ![](/img/releases/led-images/image006.jpg) | <h3>Non-compliant Charger</h3><br></br>LED 3 will blink red thrice, then off for 6 sec. The cycle repeats.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| ![](/img/releases/led-images/image008.jpg) | <h3>Charging(compatible/sufficient charger)</h3><br></br>0-5%: LED 1 will blink white and red alternatively<br></br>6–20%: LED 1 will breathe white, then off for 3 sec.<br></br>21–40%: LED 1, 2 will breathe white, then off for 3 sec<br></br>41–60%: LED 1-3 will breathe white, then off for 3 sec<br></br>61–80%: LED 1-4 will breathe white, then off for 3 sec<br></br>81–99%: LED 1-5 will breathe white, then off for 3 sec                                                                                                                                                                                 |
| ![](/img/releases/led-images/image010.jpg) | <h3>Charging(Insufficient Charger)</h3><br></br>0 - 20%: LED 1 will blink red, then off for 3 sec<br></br>21–40%: LED 1 will breathe white, 2 in red, then off for 3 sec<br></br>41–60%: LED 1-2 will breathe white, 3 in red, then off for 3 sec<br></br>61–80%: LED 1-3 will breathe white, 4 in red, then off for 3 sec<br></br>81–99%: LED 1-4 will breathe white, 5 in red, then off for 3 sec                                                                                                                                                                                                                   |
| ![](/img/releases/led-images/image012.jpg) | <h3>Charging at 100%</h3><br></br>All 5 LEDs will blink thrice, then off for 3 sec - repeated.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| ![](/img/releases/led-images/image014.gif) | <h3>Battery Status (discharging)</h3><br></br>0-5% (critical battery): LED 1 will blink red<br></br>\*The following battery patterns will be displayed in stand-by mode. In a normal discharging scenario, there will be no LED patterns on the Beltpack.<br></br>6–20%: LED 1 will blink white slowly in a breathing manner<br></br>21–40%: LED 1-2 will blink white slowly in a breathing manner<br></br>41–60%: LED 1-3 will blink white slowly in a breathing manner<br></br>61–80%: LED 1-4 will blink white slowly in a breathing manner<br></br>81–100%: LED 1-5 will blink white slowly in a breathing manner |
| ![](/img/releases/led-images/image016.jpg) | <h3>Bluetooth Pairing</h3><br></br>Simultaneously pair of 2 LEDs will turn on blue and will be executed sequentially back and forth:<br></br>LEDs 1 & 2, then LEDs 2 & 3, then LEDs 3 & 4, then LEDs 4 & 5.<br></br>Thereafter, the sequence will get reversed:<br></br>LEDs 4 & 3, then LEDs 3 & 2, then LEDs 2 & 1                                                                                                                                                                                                                                                                                                  |
| ![](/img/releases/led-images/image018.jpg) | <h3>Successful Pairing</h3><br></br>Blue color.<br></br>All 5 LEDs will be blue and on for 3 seconds.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| ![](/img/releases/led-images/image020.gif) | <h3>Unsuccessful Pairing</h3><br></br>All 5 LEDs will be blue and blink red intermittently.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| ![](/img/releases/led-images/image022.gif) | <h3>Updating</h3><br></br>White color.<br></br>LED 3 will turn on. LEDs 2 and 4 will turn on and LED 3 will turn off simultaneously. LEDs 1 and 5 will turn on while LEDs 2 and 4 turn off simultaneously. LED 3 will turn on while LEDs 1 and 5 turn off simultaneously.                                                                                                                                                                                                                                                                                                                                             |
| ![](/img/releases/led-images/image024.jpg) | <h3>Generic Success</h3><br></br>White color.<br></br>All 5 LEDs will simultaneously blink white slowly thrice. After the three blinks, there is a gap of 1 second with all 5 LEDs remaining on.                                                                                                                                                                                                                                                                                                                                                                                                                      |
| ![](/img/releases/led-images/image026.jpg) | <h3>Generic Failure / Thermal Warning</h3><br></br>Red Color.<br></br>All 5 LEDs will simultaneously blink red slowly thrice. After the three blinks, there is a gap of 1 second with all 5 LEDs remaining off.                                                                                                                                                                                                                                                                                                                                                                                                       |
| ![](/img/releases/led-images/image028.gif) | <h3>Thermal Fault</h3><br></br>LED 1 & 5 will blink red repeatedly                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ![](/img/releases/led-images/image030.gif) | <h3>Software Failure</h3><br></br>LED 1 through 3 will blink red repeatedly                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| ![](/img/releases/led-images/image032.gif) | <h3>Hardware Failure</h3><br></br>LED 3 through 5 will blink red repeatedly                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| ![](/img/releases/led-images/image034.gif) | <h3>Recovery Mode</h3><br></br>LED 2 through 4 will blink red repeatedly                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |

| ![](/img/releases/led-images/image044.gif)                                            | <h3>Charging</h3><br></br>The LED will quickly turn on and off with the color white                                          |

| ![](/img/releases/led-images/image047.gif)                                            | <h3>Firmware mismatch / Generic Failure</h3><br></br>The LED will blink repeatedly with the color red                        |
| ![](/img/releases/led-images/image049.gif)                                            | <h3>Bluetooth pairing</h3><br></br>LED will slowly turn on and off with the color blue                                       |
| ![](/img/releases/led-images/image051.gif)                                            | <h3>Successful pairing</h3><br></br>LED will remain in blue color without blinking                                           |
| ![](/img/releases/led-images/image052.gif) ![](/img/releases/led-images/image053.gif) | <h3>Unsuccessful Pairing</h3><br></br>LED will alternate between blue and red colors                                         |
