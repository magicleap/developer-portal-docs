## Overview

This example demonstrates the following features:

- image tracking
- aruco marker tracking
- barcode/QR scanning

:::note
For instructions on installing the SDK Examples Project and building this scene, follow the [**Examples Project Install Guide**](/versioned_docs/version-31-Aug-2023/guides/unity/sdk-example-scenes/sdk-install-setup.md).
:::

### Image Tracking

Show your headset an image of the following:

- deep sea
- deep space

### ArUco Tracking

Show your headset an aruco marker from the dictionary set on the MLArucoTrackerBehavior object.

You can find markers at [https://chev.me/arucogen/](https://chev.me/arucogen/).

### Barcode/QR Scanning

Show your headset a barcode/QR code. Then press and hold trigger to scan.

## Requirements

Privileges:

- CameraCapture
