# NativeFixedList



**NameSpace:** 
[MagicLeap](/versioned_docs/version-02-Aug-2023/unity-api/api/UnityEngine.XR.MagicLeap/UnityEngine.XR.MagicLeap.md) 


Has List-like semantics (Capacity and Length) using a NativeArray as the backing store. The NativeArray is never resized. This is useful for times you don't know how big the array will be, but there is a definite upper bound. This list supports duck-typed foreach Enumerator semantics.
