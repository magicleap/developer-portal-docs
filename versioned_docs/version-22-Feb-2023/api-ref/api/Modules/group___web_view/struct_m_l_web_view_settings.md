## Detailed Description

```cpp
struct MLWebViewSettings;
```

Struct to define webview initialization. 

This structure must be initialized by calling [MLWebViewSettingsInit](/versioned_docs/version-22-Feb-2023/api-ref/api/Modules/group___web_view/group___web_view.md#void-mlwebviewsettingsinit) before use.
